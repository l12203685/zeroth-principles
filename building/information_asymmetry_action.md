# 資訊不對稱 → 行動 — Information Asymmetry → Action

> 有 edge 就打，沒 edge 就等。
> 不是所有的行動都有價值——只有在你知道別人不知道的事情時，行動才有正 EV。

---

## 觀察

跨領域收集「為什麼行動」的理由，一個結構性分裂浮現：

**人們行動的理由大多錯了。**

| 領域 | 常見行動理由 | 是否有 edge |
|------|------------|------------|
| 交易 | 「大家都在買，我也要買」 | 沒有（公開資訊已定價） |
| 撲克 | 「我牌很好，應該推注」 | 不一定（牌好 ≠ range 佔優） |
| 阿瓦隆 | 「我是好人，應該主動帶隊」 | 不一定（資訊量 ≠ 資訊優勢） |
| 職涯 | 「這個機會聽起來不錯」 | 未知（沒評估資訊差異）  |
| 投資 | 「券商說這支股票有潛力」 | 沒有（券商的資訊是公開的） |

表面上，這些行動理由各自不同。但深一層看，缺陷完全一樣：**行動的依據不是「我知道別人不知道的」，而是「我有感覺」或「別人都這樣做」。**

感覺和群眾不是 edge 的來源——它們是噪音。

---

## 核心結構：Edge 是資訊差異，不是資訊量

資訊不對稱的定義很精確：**你知道，對方不知道；或你的解讀速度比對方快；或你的解讀框架比對方優。**

這三者都是 edge，但層次不同：

| Edge 類型 | 定義 | 可持續性 |
|----------|------|---------|
| 資訊獨占 | 你有他人沒有的原始資料 | 低（內線消息不合法；獨家管道會被複製） |
| 解讀速度 | 你比別人更快從公開資料得出結論 | 中（可被機器取代） |
| 解讀框架 | 你的分析框架比別人更接近現實 | 高（框架的構建是累積的，難以被複製） |

**最強的 edge 不是你知道「什麼」，而是你用什麼「框架」解讀同樣的資訊。**

這就是為什麼零式思維本身是一種 edge——不是因為它提供了別人沒有的資料，而是因為它用更接近底層機制的框架解讀所有人都能看到的現象。

---

## 為什麼：沒有 Edge 的行動是負 EV 的

行動有成本：
- 交易有手續費、滑價、資本佔用
- 撲克有時間成本和籌碼風險
- 職涯決策有沉沒成本和機會成本
- 阿瓦隆有資訊暴露的代價

當你沒有 edge 時，這些成本沒有對應的期望收益來抵消。**沒有 edge 的行動不是中性的——它是系統性的自我損耗。**

這和公設五（偏向不行動）的邏輯完全一致：「沒 edge 就等」不是消極，是避免負 EV 行動的積極選擇。

但公設二更進一步：**有 edge 時，不行動是犯罪。**

等待的機會成本在有 edge 時是真實的。如果你知道別人不知道的事情，而你選擇不行動，那個 edge 會消失——市場會調整，資訊會擴散，窗口會關閉。**資訊不對稱有時效性。** 越接近公開，edge 越薄。

## 四個領域的精確映射

### 交易：Alpha 的本質

交易市場的核心問題：**你的對手盤是誰，他們的資訊是什麼？**

散戶的最大錯誤不是選錯股——而是在沒有 edge 的情況下選股。

主動選股的隱含主張是「我比市場平均更了解這支股票的未來」。這個主張需要一個具體的 edge：

- **資訊 edge**：你掌握了市場還沒反映的訊息（法說會細節、供應鏈調查、實地訪查）
- **速度 edge**：你的處理速度比市場快（高頻交易、量化系統）
- **框架 edge**：你的分析框架比市場共識更準確（例如：你用 derivative 分析市場還在用 level 的時候）

沒有這三者之一，主動選股的期望值是負的——因為你的對手盤是整個市場，而市場的平均分析能力和資訊量都比個人高。

**正確的交易問題不是「這支股票好不好」，而是「在這個交易中，我的 edge 是什麼？如果說不出來，不進場。」**

這個問題的答案決定部位大小：edge 越大，下注越重。沒有 edge，下注為零。

### 撲克：Range Advantage 就是資訊不對稱

撲克裡的資訊不對稱是雙向的：你不知道對手的底牌，對手也不知道你的。

但「不知道對手底牌」不代表「資訊對稱」。**資訊不對稱存在於 range 層次，不是底牌層次。**

Range advantage（range 佔優）是最重要的 edge 類型：

- **高牌面上的 range 優勢**：如果你的 range 在這個牌面上有更多 top pair+ 的組合，你有 range advantage，應該持續下注施壓。
- **低牌面上的 range 優勢**：如果你是前位開池者，低牌面更有利於你的 range（因為你的 3-bet range 包含的口袋對不受低牌影響），你應該 c-bet 高頻率。

Range advantage 的計算就是資訊不對稱的量化。你知道你的整個 range，你用貝葉斯推斷對手的 range。**你對對手 range 的了解程度，減去對手對你 range 的了解程度，就是你的資訊 edge。**

沒有 range advantage 的下注是 bluff 被 call 或 value 被 fold——都是負 EV。有 range advantage 的下注提取了資訊不對稱的溢價。

**實踐：每次下注前問「我在這個 spot 的 range advantage 是正還是負？」這個問題取代「我的牌好不好？」**

### 阿瓦隆：資訊層級是結構設計

阿瓦隆的資訊結構不對稱是遊戲設計的核心：

| 角色 | 知道什麼 | 不知道什麼 |
|------|---------|----------|
| 梅林 | 所有壞人身份 | 不能暴露自己 |
| 壞人 | 彼此身份 | 梅林是誰 |
| 普通好人 | 只知道自己 | 所有人身份 |

這個資訊層級創造了三種 edge：

**梅林的 edge**：擁有完整資訊，但不能直接使用。梅林的 edge 是「在保持隱蔽的前提下最大化對好人的引導效率」。**行動原則：在邊際成本最低的時刻洩露關鍵資訊**——例如在任務投票上使用梅林的資訊，而不是在開場就明示壞人身份（那樣刺客會立刻找到梅林）。

**壞人的 edge**：彼此知道身份，可以協調。**資訊共享本身是資產**——壞人的 edge 來自協調能力，而非個別資訊量。好人沒有這個協調 edge，因此壞人的預期勝率在資訊上是結構性佔優的。

**普通好人的困境**：沒有結構性的資訊 edge，必須從行為訊號推斷。好人的 edge 只能來自解讀框架——觀察模式、識別矛盾、貝葉斯更新。這是最難的 edge，也是最可持續的 edge（因為它依賴推理能力，不依賴初始資訊量）。

**核心洞見：在阿瓦隆中，「我覺得他是壞人」不是行動依據。「我有哪個觀察讓我比其他好人更了解這個人是壞人」才是行動依據。** 沒有獨立觀察，就跟著梅林的訊號走；梅林訊號不明確，就不主動下結論。

### 職涯：解讀框架是最可持續的邊際

職涯決策中的資訊不對稱最難識別，因為「職涯 edge」是個模糊的概念。

但拆解後，它的結構很清楚：

**你的職涯 edge 存在於你能做、其他人做不了、市場尚未充分定價你這個能力的交集。**

| 條件 | 說明 |
|------|------|
| 你能做 | 技能確實存在 |
| 他人做不了 | 稀缺性 |
| 市場尚未定價 | 資訊不對稱尚存 |

當市場充分定價你的稀缺能力時，edge 消失。這正是「學了 AI 工具就加薪」的窗口正在關閉的原因——當所有人都會用 AI 工具時，稀缺性消失，市場定價調整，edge 歸零。

**職涯的資訊不對稱有兩個方向需要監測：**

1. **市場低估你的方向**：你的技能還沒被充分定價，應該行動（轉職、加薪談判、接案）。
2. **你高估自己的方向**：你以為有 edge，但市場已經充分定價了。這時行動是負 EV 的。

**多數人犯的是第二類錯誤——在沒有 edge 的職涯決策上行動，用「機會看起來不錯」取代「我在這個機會上的 edge 是什麼」。**

## 資訊不對稱的時效性

Edge 不是靜態的。它有生命週期：

```
發現 → 獨占窗口 → 擴散 → 定價 → edge 歸零
```

**在「獨占窗口」行動，而不是在「擴散」之後行動。**

這個生命週期在不同領域的速度不同：

| 領域 | Edge 時效 |
|------|----------|
| 高頻交易 | 毫秒級 |
| 日內策略 | 小時級 |
| 基本面交易 | 週到月 |
| 職涯技能 | 年到十年 |
| 框架思維 | 極長（框架構建需要時間，難以複製） |

這意味著**行動的緊迫性跟 edge 的時效成反比**：時效越短，行動越緊迫。框架層的 edge 可以慢慢構建和行動；資料層的 edge 必須立即行動，否則窗口關閉。

## 零式根源：Edge 的本質是訊號雜訊比

資訊不對稱的深層結構是訊號雜訊比的差異。

所有人都在接收訊號，但訊號中埋著雜訊。你的 edge 來自：

**你的訊號雜訊比 > 市場平均的訊號雜訊比**

提高訊號雜訊比的方法只有兩個：
1. **更好的資料來源**（靠近第一手資訊）
2. **更好的過濾框架**（識別哪些是訊號，哪些是雜訊）

第一個方法有邊際遞減——資料來源能有多好，有結構性上限（內線消息違法；第一手調查需要巨大的成本）。第二個方法沒有上限——框架可以無限精煉，而且一旦內化，複製成本接近零。

**這就是為什麼零式思維是可持續 edge 的來源：它不是一組規則，而是一個持續演化的解讀框架。每次應用，框架更精準；框架更精準，訊號雜訊比更高；訊號雜訊比更高，edge 更大。這是正向遞迴的。**

## 與其他公設的交互

**→ 公設一（Derivative > Level）：** 最深的資訊不對稱不是「我知道現在的狀態」（level），而是「我看到狀態的變化方向」（derivative）。Level 是公開資訊，derivative 是需要計算和框架才能提取的資訊。你的 derivative 分析能力就是你的資訊 edge。

**→ 公設三（後設策略 > 策略）：** 資訊不對稱存在於兩個層次。策略層：「我對這個交易有 edge 嗎？」後設層：「我有系統性地識別自己什麼時候有 edge 的能力嗎？」後設層的 edge 比策略層更珍貴——它讓你在任何新情境下都能快速判斷是否行動。

**→ 公設四（群眾反向）：** 群眾的行動是公開資訊。「群眾在買」這件事本身沒有 edge——人人都知道。但「我能計算群眾的偏差方向和速度」是資訊不對稱——因為大多數人不做這個計算。Population exploit 的執行依賴的正是資訊不對稱：你看到群眾偏差，他們自己沒看到。

**→ 公設五（偏向不行動）：** 公設二和五是一體兩面。公設五定義了不行動的條件（沒 edge）；公設二定義了行動的條件（有 edge）。兩者合起來形成完整的決策函數：**f(edge) → 有 edge = 行動，無 edge = 等待。** 這個函數不是「有時行動，有時等待」的模糊原則——它是精確的二元開關，由 edge 是否存在決定。

## 實踐原則

**個人決策層：**

- **每次行動前明確說出你的 edge。** 「我在這個決策中的 edge 是什麼？」如果說不出具體的答案，不行動。「感覺好」「大家都在做」「看起來不錯」都不是 edge 的描述。
- **區分資訊量和資訊差異。** 你讀了很多研究報告 ≠ 你有 edge。Edge 是你的資訊減去市場已定價的資訊。如果市場知道你知道的所有事情，edge 為零。
- **監測你的 edge 有效期。** 你在職涯某個技能上的 edge 還有多久？你在某個交易策略上的 edge 是否已經被複製？Edge 消失時，停止依賴它，尋找新的 edge。

**系統設計層：**

- **建立 edge 登記系統。** 在每個交易進場前記錄：「我的 edge 是什麼？這個 edge 的來源是資訊獨占、速度還是框架？預計有效期多長？」這個記錄讓你事後驗證——你以為的 edge 是真實的嗎？
- **部位大小正比於 edge 強度。** Kelly Criterion 的核心邏輯：知道 edge 的大小，下注的大小跟著走。Edge 大 = 重倉。Edge 小 = 輕倉。Edge 零 = 不進場。不做等權重的倉位分配，那是在承認你無法區分 edge 的大小。
- **反向驗證你的 edge。** 問「如果我錯了，最可能的原因是什麼？」好的 edge 能回答這個問題。如果你找不到「在什麼條件下這個 edge 不存在」，你的 edge 可能是幻覺。

## 結論

有 edge 就打，沒 edge 就等——這兩句話合在一起才是公設二的完整形式。

多數人只記得「有機會就要抓住」，忘了這個原則的前提是「機會 = 你有別人沒有的 edge」。沒有 edge 的機會不是機會，是陷阱。

**資訊不對稱是行動的必要條件，不是充分條件。** 你還需要 edge 的時效窗口、正確的倉位大小、以及執行能力。但如果連必要條件都沒有——如果你說不清楚你的 edge 是什麼——一切後續都是在沙上建屋。

更深一層：主動尋找和構建 edge，才是長期複利的來源。框架層的 edge 是自我強化的：越思考，框架越精準；框架越精準，識別 edge 的能力越強；識別能力越強，行動質量越高；行動質量越高，結果回饋越豐富，框架持續精煉。

**原則：行動的唯一正當理由是資訊不對稱。有 edge，全力出擊；沒 edge，耐心等待。等待不是浪費——它是累積 edge 的過程。最危險的狀態不是「沒機會」，而是「在沒有 edge 的情況下行動」。**
