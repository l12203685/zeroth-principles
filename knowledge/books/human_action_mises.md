### TL;DR (≤120字)
奧地利學派鉅著,Mises 晚年總結作。核心:(1) 行為學 (praxeology) — 人類行為永遠是為達目的而選擇,這是先驗真理,不能實證反駁;(2) 經濟計算 — 市場價格是資源配置的唯一理性機制,社會主義因無價格而必然失敗 (Mises 1920 論文「社會主義經濟計算問題」的完整展開);(3) 通膨是貨幣擴張的必然結果,只要政府壟斷貨幣,通膨是長期宿命。長 1000 頁,結構嚴密,對所有干預主義作徹底否定。

### 核心本質 (3-5 條, 每條 50-120字)
1. **行為學先驗基礎 (Ch.1-3)** — 「人類行為」是有意圖選擇以達目的的行動。這不需要實證驗證 — 否認行為存在本身就是一個行為。由此先驗推出:(a) 主觀價值論 (b) 邊際效用 (c) 時間偏好 (d) 不確定性下的選擇。所有經濟學可從此基礎推演。與 Popper 的實證主義對立。
2. **經濟計算論證 (Ch.11-13, 23-26)** — 無市場 + 無私有生產資料 = 無交換比例 = 無價格 = 無法計算何種生產方式最有效。社會主義國家無法知道「用鋼造拖拉機 vs 用鋼造坦克」哪個更滿足需求。實證:蘇聯計劃失敗、中國改革開放的成功、印度 1947-1991 的計劃停滯。
3. **通膨的必然性 (Ch.28)** — 政府壟斷貨幣 + 財政壓力 = 通膨偏差。不是「意外」或「政策錯誤」,是結構內生。穩定貨幣需要外部紀律:金本位、獨立央行、或競爭貨幣 (Mises 1928 首倡 "free banking")。Bitcoin 是 Mises 思想的數位實現。
4. **企業家角色 (Ch.14-15)** — 企業家 = 在不確定未來中判斷,承擔盈虧的人。其功能是「比市場更快發現低估機會並糾正」。不是 arbitrage (無風險套利) 而是 imagination + bet。此概念被 Schumpeter、Kirzner 擴展,成為奧地利學派核心。
5. **干預主義死路 (Ch.27-35)** — 每次政府干預 (價格管制、最低工資、關稅、補貼) 都產生副作用,副作用驅動更多干預,最終走向全面計劃。歷史上沒有「中間路線」能穩定存在。關鍵警告:小干預不停留於小,會自動擴大。

### 可用戰術/策略
- **Praxeology 自檢** — 任何經濟分析前,先問:這個情境下行為者的主觀目的是什麼? 不要假設「理性行為者追求利潤最大」(過於簡化),而是「在其認知下追求其目的」。對市場心理分析更精準。
- **經濟計算可行性檢驗** — 任何新經濟設計 (DAO, 國有企業, 計畫組織):是否有價格機制? 是否有盈虧回饋? 是否有明確產權? 三問皆否 → 長期失敗。
- **通膨長期對沖** — Mises 框架下,法幣長期必通膨。資產配置應含至少 10-20% 硬資產 (黃金、BTC、土地、生產性資產) 作為 Mises 式對沖。
- **企業家 vs 管理者區別** — 組織設計:識別誰在承擔真實風險 (企業家) vs 誰在執行既定流程 (管理者)。薪酬結構應反映 — 企業家分紅,管理者薪水。Edward 自營系統的「自己擔盤、自己承受盈虧」正是 Mises 意義下的企業家行動。
- **干預一致性檢驗** — 分析任何政策包:是否每項都解決前一項副作用? 是 → 進入干預螺旋,時間不利。Ex: 央行 ZIRP → 資產泡沫 → 打房 → 供給萎縮 → 通膨再起 → QT → 金融不穩 → 再 QE...

### 盲點 / 反例 / 適用邊界
- **Praxeology 方法論飽受批評** — Mises 反實證主義過於絕對。現代經濟學傾向混合 — 先驗演繹提供結構,實證檢定細節。 Mises 純先驗論易陷自證不足 (Friedman 1953 批判)。
- **Austrian 對 Keynes 反對過全** — Mises 反對任何財政刺激、任何貨幣擴張;但 2020 疫情時期,若依 Mises 不刺激,可能短期衰退極深。現代奧派 (Mark Skousen) 承認有限度的緊急介入。
- **制度設計缺乏現代實驗** — Mises 未預見中央銀行數位貨幣 (CBDC)、加密貨幣、DeFi、Al 平台經濟等新工具的出現。他的框架強,但細節需更新。
- **社會福利論證偏弱 (Ch.28-30)** — Mises 從「效率」否定福利國,但未充分處理「效率 vs 倫理/公平」的張力。Hayek 1960 年後在此議題上更細緻。
- **加密泡沫的 Mises 解釋困難** — Bitcoin 看似是 Mises 理念的勝利,但 2021-22 crypto 狂熱顯示即使沒有政府干預,私部門也會產生泡沫 (違反 Mises 部分樂觀)。可能需與 Minsky 結合解釋。

### 與 Edward 既有知識的連結
- 對齊 `road_to_serfdom_hayek` 和 `constitution_of_liberty_hayek`:Mises 是 Hayek 的老師,兩人思想一脈。Hayek 比 Mises 更制度化、政策化;Mises 更先驗、激進。
- 連結零式 `derivative_over_level`:Mises 的「行為 = 在邊際上選擇」正是 derivative 思維 — 不看 stock 看 flow,不看 level 看 change。Edward 的日常決策紀律完全奧派。
- 深化 B1 加密策略的哲學基礎:Mises 1920 "Socialism" + 1949 "Human Action" + 1928 "free banking" 構成 Bitcoin 的思想源頭。Edward 作為加密參與者,實是 Mises 理念的活體執行。
- 對 DNA 決策分權:Mises 的 praxeology + 主觀價值論直接對應 Edward-Koko substance-lead/form-defer — 我不假設知道她的「最佳選擇」,只給 options 讓她按自己的主觀價值排序。
- 衝突於 Keynesian 乘數和 `stabilizing_unstable_economy_minsky`:Mises 認為 Minsky 式大政府大央行正是問題根源,非解決方案。Edward 需在宏觀事件中識別「是 Mises 對 (政府扭曲) 還是 Minsky 對 (內生脆弱)」,兩者可能同時為真。
- 延伸 DNA 人格 OCEAN:Mises 寫作展現極致的 "high openness + high neuroticism"( 對自由的焦慮)。Edward 的座標靠近 Hayek (平衡) 而非 Mises (激進純粹派),但保持對 Mises 洞察的深度尊重。
