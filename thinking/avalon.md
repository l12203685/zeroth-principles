# 阿瓦隆推理方法論

> 阿瓦隆是不完整資訊下推理的沙箱。
> 所有從這裡發展的工具，都可以直接映射到交易、投資、人際判斷。

---

## 核心公式

$$Victory = Logic \times Trust + Deception \times Chaos$$

- 好人的路徑：用邏輯建立信任，壓縮壞人的混亂空間
- 壞人的路徑：用欺騙製造混亂，讓邏輯失效
- **這不是比誰更聰明，而是比誰更能控制信息熵**

---

## 遊戲結構

**勝負條件（搜索空間邊界）**：

| 陣營 | 勝利條件 |
|------|---------|
| 好人 | 完成 3 個任務（梅林存活） |
| 壞人 | 破壞 3 個任務，或連 5 輪拒隊，或刺殺梅林 |

**關鍵不對稱**：
- 壞人知道所有壞人是誰，好人只知道自己
- 梅林知道壞人，但暴露即輸
- 派西看到假梅林（壞人可佔位），不確定性加倍

**每輪結構**：
```
選隊（隊長）→ 派票（全員）→ 任務（隊員）→ 資訊更新（全員）
```

## GTO vs 剝削策略映射

阿瓦隆的策略均衡本質：

| 概念 | 撲克 GTO | 阿瓦隆對應 |
|------|---------|-----------|
| 不可剝削 | 詐唬與價值頻率平衡 | 好人行為不透露身份模式 |
| 剝削策略 | 針對對手漏洞調整 | 壞人針對特定好人信任關係滲透 |
| Range | 所有可能手牌集合 | 任一玩家的身份假說空間 |
| 熵壓縮 | 縮小對手range | 每次觀測後降低身份不確定性 |

**結論**：
- 壞人的最優策略不是「假裝好人」，而是維持全局信念熵高位，讓好人互相懷疑
- 好人的最優反制是製造可觀測的資訊事件，強制壞人暴露 EV 選擇

## 六公理

| # | 公理 | 含義 |
|---|------|------|
| 1 | **一致性** | 所有推論必須和已知事實相符，不能自相矛盾 |
| 2 | **廣泛性** | 考慮所有可能的假設，不過早排除低概率情境 |
| 3 | **精簡性** | 解釋力相同時，假設最少的模型優先（奧卡姆剃刀）|
| 4 | **優勢性** | 優先考慮對特定陣營有利的解釋 |
| 5 | **可證偽性** | 推論必須能被未來觀測推翻或支持 |
| 6 | **自我修正** | 新資訊進來後主動更新信念，不捍衛舊推論 |

**第六條是核心**：能自我修正的推理者，最終會收斂到正確答案。

## 三層邏輯架構

```
鐵邏輯（必然成立）
  → 如果 A 是壞人，這個結論無論如何都成立
  → 最高權重，直接縮小搜索空間

正邏輯（概率加分）
  → 這個行為符合壞人的期望值最大化
  → 中等權重，累積後影響信念

反邏輯（反向排除）
  → 這個行為如果是壞人做的，反而不符合 EV
  → 用來排除假說，不用來確認
```

## 區間派票法（Bayesian 信念更新）

每位玩家維護一個信念區間：

$$[P(\text{好人}), P(\text{壞人})], \quad P(\text{好人}) + P(\text{壞人}) = 1$$

每次觀測事件觸發更新：

$$P(\text{壞人} | \text{行為}) = \frac{P(\text{行為} | \text{壞人}) \cdot P(\text{壞人})}{P(\text{行為})}$$

**可觀測的觸發點**：
- 投票 for / against 隊伍
- 任務 pass / fail
- 發言內容是否和已知資訊矛盾
- 沉默（不發言也是資訊）

**信號可信度排序**（由高至低）：

$$\text{任務結果} > \text{湖中女妖} > \text{派票黑白球} > \text{狀態發言}$$

- 任務結果是硬信息：pass/fail 直接約束壞人人數的上下界
- 湖中女妖揭露真實身份，但被揭露者知曉 → 後續行為受此影響
- 派票是行為，比話語難以偽裝
- 狀態發言噪音最高，EV 分析權重最低

## 自我修正的循環流程

推理不是單次輸出，而是一個持續迭代的狀態機：

```
假設 → 派票（行動）→ 觀測結果（驗證）→ 修正 → 新假設 → ...
```

**觸發修正的條件**（對應信號可信度排序）：
1. 任務 pass / fail → 直接約束壞人人數，強制更新
2. 湖中女妖揭露身份 → 全局信念重新定錨
3. 他人派票反饋 → 提取他人視角的隱含假設
4. 發言（最低權重）

**修正 vs 前後矛盾**：

| | 自我修正 | 前後矛盾 |
|---|---|---|
| 觸發條件 | 有（任務結果/湖中/派票） | 無 |
| 第一次派票 | 吃資訊（有依據） | 能力派（超出自身資訊） |

- **自我修正**：「上一局紅燈，所以我更新了我的假設」
- **前後矛盾**：「沒有任何新資訊，但我改了」→ 壞人 tell

核心洞察：**修正需要契機。沒有契機的改變是偽裝，不是學習。**

## 反邏輯：高階欺騙模式

**正邏輯**（80-85% 情況適用）：建立在「壞人行動以最快達成三紅為目標」的假設上推理，是可信賴的基本功。

**反邏輯**：透過多數人以正邏輯思考的路線，刻意做出「表面上低效益」的行動，讓對手在以正邏輯分析時產生視角盲點。

- 反邏輯的真實效益，發生在「場上所有人用正邏輯推理後的後續發酵」
- 這不是隨機亂打，而是對正邏輯推理路徑的主動干擾
- **識別方式**：對方做出「表面低效益但有特定解讀優勢」的行動 → 可能是反邏輯

**出者恆出，藏者恆藏**：行為一致性是識別相認壞人的強信號。相認壞人傾向於讓彼此輪流出失敗以最大化效益；若兩局都不出，代表未相認或打反邏輯。

## 任務只能證偽，不能證實

**波普爾原則在阿瓦隆的應用**：

任務結果開紅 → 強制縮小假說空間（至少一個壞人在此隊）
任務結果開藍 → 無法確認此隊全為好人（壞人可能開白）

這意味著：
- 藍燈 = 「目前假設未被推翻」，不等於「假設正確」
- 所有推理都是暫定假設，等待更多紅燈事件收斂

**第一次派票原則**：
- 「吃資訊型」（基於已知資訊推測）→ 修正容易被認為是學習
- 「能力派」（超前自身資訊範圍宣稱確定）→ 修正容易被認為是壞人矛盾

## 排除優於指認

在多人局中直接找出壞人很難，但確認好人相對容易。

$$P(\text{Bad}) = \frac{\text{Total\_Bad}}{\text{Total\_Players} - \text{Confirmed\_Good}}$$

先確認誰是好人，壞人的搜索範圍自然縮小。鐵好人每多確認一個，壞人被稀釋一次。

**深水狼策略**（壞人最優隱藏模式）：
- 前期完全模仿好人行為，甚至投票給好人隊伍累積信任
- 只在賽點（第3個任務）背刺
- 反制方法：觀察「從不被懷疑的人」——長期零壓力本身是資訊

**梅林困境 + 派西維爾責任**：
- 梅林知道真相但不能說破 → 必須用「引導式提問」而非「宣告式結論」傳遞資訊
- 派西維爾必須主動裝作像梅林，吸引刺客砲火保護真正梅林
- 設計層面：角色不對稱創造信息博弈，雙方都在解讀對方的偽裝成本

## 關鍵洞察

**不需要「確定」，只需要「夠高的概率」**

- 壞人的最優策略：維持全局信念熵高位（製造混亂）
- 好人的最優策略：每次觀測後降低熵（製造清晰）
- 推理能力的差異，最終體現在「誰能更快壓縮不確定性」

## 廣泛性 vs 平衡策略

| 策略 | 定義 | 目的 |
|------|------|------|
| **廣泛性（遊戲分支）** | 讓未來可說的故事變多，包含多種可能走向 | 增加可變性，能力者有空間展現資訊 |
| **平衡策略** | 打保險牌，避免所有人一起崩盤 | 守住底線，防止系統性誤解 |

追求廣泛性 ≠ 平衡策略。廣泛性是主動擴展可能性；平衡是被動守住損失上限。

**映射**：在不完整資訊博弈中，保持「故事線多」 = 讓對手的搜索空間更難收斂。好人應追求廣泛性，壞人應壓縮好人的廣泛性。

## 自我暴露原則

**識別自己是否正在洩露身份**：

若你已在「場上認為有壞的區間」內，卻繼續派選同區間的其他人 → 等同默認「那個人比我好，我才是壞人」。

**檢查點**：
- 我現在派的這個人，是否在場上所有人都認為有壞的區間？
- 若是，繼續派他 = 間接承認自己是該區間中更壞的那個

這是一種「派票路徑暴露」——不是通過直接宣告，而是通過選擇邏輯被推算出身份。

## 派票意識五問

每次派票前，能清楚回答以下五個問題才算是「有意識的派票」：

1. **前提**是什麼？（基於哪些已知資訊？）
2. **選了誰**？（名單是什麼？）
3. **為什麼**選？（每個人入選的理由？）
4. **表達了什麼**？（其他人看到這個名單會推論什麼？）
5. **希望達到什麼效果**？（這個派票的目的是什麼？）

無法回答第4和第5題 = 無意識派票，場上雜訊而非資訊。

## 多壞人進任務：位階法

當多個相認壞人同時進入任務時，誰出失敗的規則：

- **位階法**：依照位置 / 場上信任度決定誰「藏」誰「出」
- **後進者出**：最後進入隊伍的壞人出失敗（在場上更不容易被指向）
- **出者恆出，藏者恆藏**：已暴露的出失敗，未暴露的繼續隱藏

**反制方法（好人視角）**：觀察「從來沒被懷疑、長期零壓力的玩家」——長期無壓力本身是非對稱信號。

## 多重假設謬誤

**定義**：從兩個不同假設各自推導出相同結論，即使結論正確，仍是錯誤的思考過程。

範例（錯誤）：
- 假設 A 好 → B 偏好
- 假設 A 壞 → E 偏好
- 結論：選 B 和 E

問題：B 偏好依賴「A 好」成立；E 偏好依賴「A 壞」成立。兩個前提互斥，不能同時成立。

**正確做法**：選定一個假設，一條邏輯線走到底。錯了可以被修正，但邏輯必須自洽。

**跨域映射**：在投資決策中，「無論升息還是降息股市都會漲」這種邏輯等同多重假設謬誤 — 結論來自兩個互斥前提，不是邏輯，是逃避。

## 反劣勢性（比找最優解更穩健）

在阿瓦隆中，找到「最優派票」很難，但找到「明顯劣勢的派票」相對容易。

**反劣勢性策略**：
1. 列出所有可能故事線
2. 找出在多數故事線下表現都差的選擇 → 劣勢策略
3. 排除劣勢策略，在剩餘選項中選擇

「找出最差的並排除」比「找出最好的並選擇」更不依賴單一假設的準確性。

**跨域映射**：投資組合選擇中，消去法（排除高費率、高相關性、低流動性的選擇）比直接尋找最優組合更穩健。

## 積極型 vs 保守型派票

| 類型 | 定義 | 優勢 | 風險 |
|------|------|------|------|
| **積極型** | 在不確定區間中做出明確選擇，嘗試分辨具體人選 | 信息解析度高 | 派錯暴露自己的局限 |
| **保守型** | 不試圖分辨，保持在合理解釋範圍內 | 低暴露風險 | 信息收益低 |

**原則**：「選一條假設走下去」— 結論不一定要對，但邏輯必須一致。錯了只要邏輯被接受，仍可以被後續玩家修正和利用。

## 信息只通過信任渠道傳播

**梅林困境的本質**：梅林掌握最完整的信息，但信息影響力 = 0，如果場上沒有人信任梅林。

**原則**：**先建立可信度，信息才能傳播。**

- 梅林必須先讓人「認下是好人」，資訊才能被後續玩家採用
- 信息本身不會自動傳播；接收者的信任是傳播的前提

**跨域映射**：
- 職場：你的建議只有在老闆信任你之後才有影響力
- 市場：分析師的觀點只在其聲譽建立後才移動資金
- 談判：資訊優勢的變現需要信任作為通道

## 信息釋放的代價原則

每次異常票（內黑/外白）釋放信息時，必然犧牲某些東西：

| 異常票 | 釋放的信息 | 代價 |
|--------|---------|------|
| **內黑** | 「我不想和某些人出任務」 | 被選概率降低；可能被誤解 |
| **外白** | 「我贊成這組，想改變尾家」 | 暴露立場；梅林刺殺風險上升 |

**通用原則**：在博弈情境中，**信息從不是免費的**。釋放信息 = 縮小對方的不確定性，同時也縮小自己的空間。

**交叉應用**（交易/投資）：公開部位方向 = 讓對手知道你的壓力點。信息釋放的時機和代價需要同等計算。

## 映射到其他領域

**量化交易**：
- 每根 K 棒是一次觀測，更新你對趨勢的信念
- 停損 = 「這個假說被推翻了，退出並重置」

**人際判斷**：
- 行為和話語不一致時，行為的貝葉斯權重更高
- 「他說了這句話，在哪種假說下是 EV 最大的行為？」

**職場**：
- 「公司說的」vs「公司做的」→ 後者是真實信號
- 升遷決策的 prior：這公司歷史上升的是什麼類型的人？
