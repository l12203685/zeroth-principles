# 阿瓦隆推理方法論

> 阿瓦隆是不完整資訊下推理的沙箱。
> 所有從這裡發展的工具，都可以直接映射到交易、投資、人際判斷。

---

## 六公理

| # | 公理 | 含義 |
|---|------|------|
| 1 | **一致性** | 所有推論必須和已知事實相符，不能自相矛盾 |
| 2 | **廣泛性** | 考慮所有可能的假設，不過早排除低概率情境 |
| 3 | **精簡性** | 解釋力相同時，假設最少的模型優先（奧卡姆剃刀）|
| 4 | **優勢性** | 優先考慮對特定陣營有利的解釋 |
| 5 | **可證偽性** | 推論必須能被未來觀測推翻或支持 |
| 6 | **自我修正** | 新資訊進來後主動更新信念，不捍衛舊推論 |

**第六條是核心**：能自我修正的推理者，最終會收斂到正確答案。

## 三層邏輯架構

```
鐵邏輯（必然成立）
  → 如果 A 是壞人，這個結論無論如何都成立
  → 最高權重，直接縮小搜索空間

正邏輯（概率加分）
  → 這個行為符合壞人的期望值最大化
  → 中等權重，累積後影響信念

反邏輯（反向排除）
  → 這個行為如果是壞人做的，反而不符合 EV
  → 用來排除假說，不用來確認
```

## 區間派票法（Bayesian 信念更新）

每位玩家維護一個信念區間：

$$[P(\text{好人}), P(\text{壞人})], \quad P(\text{好人}) + P(\text{壞人}) = 1$$

每次觀測事件觸發更新：

$$P(\text{壞人} | \text{行為}) = \frac{P(\text{行為} | \text{壞人}) \cdot P(\text{壞人})}{P(\text{行為})}$$

**可觀測的觸發點**：
- 投票 for / against 隊伍
- 任務 pass / fail
- 發言內容是否和已知資訊矛盾
- 沉默（不發言也是資訊）

## 關鍵洞察

**不需要「確定」，只需要「夠高的概率」**

- 壞人的最優策略：維持全局信念熵高位（製造混亂）
- 好人的最優策略：每次觀測後降低熵（製造清晰）
- 推理能力的差異，最終體現在「誰能更快壓縮不確定性」

## 映射到其他領域

**量化交易**：
- 每根 K 棒是一次觀測，更新你對趨勢的信念
- 停損 = 「這個假說被推翻了，退出並重置」

**人際判斷**：
- 行為和話語不一致時，行為的貝葉斯權重更高
- 「他說了這句話，在哪種假說下是 EV 最大的行為？」

**職場**：
- 「公司說的」vs「公司做的」→ 後者是真實信號
- 升遷決策的 prior：這公司歷史上升的是什麼類型的人？

## 進階洞察（wiki 文章萃取 2026-04-06）

### 三層思考上限
第一層不能錯。大多數人停在第一層。三層是資訊傳遞的接收極限，超過無效。
→ 不是「想越多層越好」，而是「在對方能接收的層數內傳遞有效資訊」。
→ 交易映射：策略複雜度超過自己能管理的層數 = 過擬合。

### 遞迴增強閉環
$$I_{t+1} = I_t \cup E(f(I_t))$$
輸出即輸入。每次決策的執行結果回饋修正下一輪決策參數。
→ 交易映射：ATR 自適應停損、MAE/MFE 回饋修正策略參數。

### 勝利公式
$$Victory = Logic \times Trust + Deception \times Chaos$$
好人最優：最大化邏輯和信任。壞人最優：最大化欺騙和混亂。兩方策略不對稱。
→ 交易映射：做多方靠基本面和趨勢（邏輯），做空方靠恐慌和資訊不對稱（混亂）。

### 資訊超前層差
「比大家快半步的是老手，比老手快的是梅林，比梅林快的是壞人。」
用資訊超前程度判斷動機和身份。
→ 交易映射：知情交易者的訊號比散戶快 → 觀察誰在你之前行動。

### 14 種具名謬誤（精選）
1. 「解決不了黑球，就解決開黑的人」— 攻擊信使 = 系統思維缺失。→ 交易：怪市場而非檢討策略。
2. 「鐵頭堅持完全不修正」— 停損拒絕 = 凹單。
3. 「過度假設」— 過擬合。用太多假設解釋少量觀測。
4. 「認為別人有讀心術」— 高估對手的資訊層級。
5. 「容錯率 vs 死採」— 選期望值正且下行風險有限的方案，不選全押。
