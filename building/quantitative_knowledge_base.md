## 一、投資本質四大定律

| # | 定律 | 核心 |
|---|------|------|
| 1 | 市場不可能三角 | 低風險+高報酬+可規模化，宣稱三者兼具 = 結構性謊言 |
| 2 | 必然事件原理 | 投資只能建立在必然發生的結構上 |

行動優先序：資產配置 > 標的選擇 > 報酬預測

## 二、交易系統規範（具體數字）

| 參數 | 值 |
|------|-----|
| 每日最大風險 | 3% |
| 單筆風險 | 1%~1.5% |
| 每日最大筆數 | 5（動能 2 + 趨勢 3）|
| 上限口數 | 50 口大台 |
| 順勢賺賠比 | ≥ 1:3 |
| 逆勢賺賠比 | 1:1.1~1.2（靠高勝率）|

### 菱形加碼 [2,3,3,2]
- 母單 30%（停損遠）→ 趨勢單 40%（拉回進場）→ 動能單 30%（停損緊 <10點）

### 停損公式
- ASL = `3% × (m/v) / (L×T) × Range`
- RSL < `0.2 × Range`
- ATR 資金管理：`Money_Mgnt_ATR = r × N × ATR + (1-r) × Const`

### 月停損機制
記錄月初淨值，若本月虧損 > `monthlevel` 或月內回撤 > `maxDDlevel` → 強制平倉。非可選，是必要保險層。

## 三、MAE/MFE 分析體系

John Sweeney (1983-1997) → Kelly (1956) → Thorp (1969) → Vince (1995) → MAE 動態停損

### 核心洞見
1. 順勢交易的 MFE 發生在 MAE **之後**（先被套再賺）；逆勢相反
2. 停損設在「勝手交易 MAE frontier」邊界之外 — 設太緊把勝手變敗手
3. 多空分拆四類：多勝/多敗/空勝/空敗
4. Edge-Ratio = MFE/MAE，可跨策略比較

### PnL Count 分類
- Q1（大SL，逆勢）vs Q2（小SL，順勢靠少數大獲利支撐）

## 五、選擇權 Greeks + 定價

| Greek | 實務用法 |
|-------|----------|
| Delta | 買方選高 Delta；賣方 Delta-Neutral 對沖 |
| Gamma | 近到期激增，賣方風險急升 |
| Theta | 賣方正值（賺時間）；買方負值（以時間換空間）|
| Vega | IV 上升有利買方 |

### Izraylevich 三層框架
策略設計（Delta-Neutral/部分方向性）→ 多目標最優化（Pareto, robustness test）→ 風險管理（VaR, Index Delta）→ 資本配置

## 六、波動率模型

| 年代 | 模型 | 特徵 |
|------|------|------|
| 1970 | 時變波動率 | clustering, fat tail, leptokurtosis |
| 1993 | Heston SV | 波動率均值回歸 + 與標的相關 |
| 1990s | GARCH(1,1) | 實證驗證最穩健 |
| 2000s | CARR (Range) | High-Low 替代 close-close |
| 利率 | SABR | smile/skew fitting |

**關鍵**：LSTM 預測股價不優於 ARIMA（Spectral Bias：優先擬合低頻 = MA）

## 八、跨主題共識

1. **期望值 > 勝率**：所有策略以 EV 為核心
2. **停損由波動結構決定**（MAE/ATR），非固定點數
3. **IV > HV** 作為選擇權定價參考
4. **ML 無顯著優勢**：ARIMA/GARCH 仍是 baseline
5. **月停損 = 必要保險層**（非可選）
6. **效率前緣最優化不可靠**（歷史共變異數不穩定）
