# 量化交易系統設計哲學

## 核心原則

> 一致性 > 精確性。
> 存活率 > 報酬率。

**投資本質三判準**：任何配置決策必須同時通過三項篩選：
1. **必然性** — 報酬來源是否建立於必然發生的結構？
2. **規模化** — 機會是否支持大規模複製且不自我毀滅？
3. **存活性** — 最壞情境下系統是否仍可繼續運作？

---

## 六大投資第一性定律

這六條是物理層的限制，不隨市場好壞而改變：

| # | 定律 | 定義 | 結論 |
|---|------|------|------|
| 1 | **市場不可能三角** | 低風險/高報酬/可規模化，三者最多取其二 | 宣稱三者兼具 = 結構性謊言 |
| 2 | **必然事件原理** | 投資僅能建立於必然發生之結構 | 做多人類生產力（指數）長期必勝 |
| 3 | **報酬不對稱原理** | 虧損50%需獲利100%才能回本 | 避開致命虧損遠比追求暴利重要 |
| 4 | **擇時無效定律** | 市場短期波動近似隨機漫步，長期擇時成功率 ≈ 50% | Time in the Market > Timing the Market |
| 5 | **階梯式成長模型** | 財富積累不是線性，而是階段跳躍 | ①勞務累積本金 → ②資產報酬 > 勞務 → ③現金流覆蓋支出 |
| 6 | **槓桿生存原則** | 槓桿放大波動，不改變期望值方向 | $L_{safe} < \frac{1}{MaxDD}$；現金流斷裂才是槓桿的殺手 |

**四大上位結構定律（原理層）**：

| 定律 | 定義 | 結論 |
|------|------|------|
| **反脆弱積累** | 長期報酬上限由系統存活能力決定 | 長期勝率源於不被淘汰，不是贏最多 |
| **不確定性守恆** | 風險不會消失，只會轉移 | 核心在於選擇承擔哪種風險 |

## 交易憲法（邊界條件，不是建議）

```
不追漲 / 不攤平
單筆虧損上限：總資產 2%
停損設置後不移動
年度 MDD 上限：30%
```

違反任何一條 → 停下來問「發生了什麼」，不是繼續操作。

## 資金配置：防禦型半Kelly

Kelly 公式給出的是「期望值最大化」的倉位。但人不是期望值機器。

**問題**：Kelly 全倉時，單筆損失會超過心理閾值 → 執行偏差增加 → 實際績效低於理論。

**解決**：使用 0.5× Kelly（半Kelly）

$$f^* = \frac{bp - q}{b} \times 0.5$$

**生理無感閾值**：
- 統計觀察：單筆損失超過總資產 2% 時，執行偏差率顯著上升
- **2% 是心理邊界，不是數學最優**
- 交易憲法的「單筆 ≤2%」是半Kelly + 生理閾值的雙重約束收斂點

**實際意涵**：
- 不要問「最大化報酬的倉位是多少」
- 要問「在什麼倉位下，我能穩定執行系統100次」
- 答案永遠小於 Kelly 全倉

## 市場狀態機

市場有三種狀態，對應不同操作模式：

| 狀態 | 觸發條件 | 操作 |
|------|---------|------|
| 攻擊 | 月線+季線多頭 + ATR 低 | 高 Beta 加碼 |
| 防禦 | 月線空頭 + ATR 高 | 避險 / CTA |
| 中性 | 訊號模糊 | 等權持倉，等待清晰 |

**「不確定的時候，站回中性。」**

### Entropy 訊號過濾

市場雜訊程度可以用資訊熵（Shannon Entropy）量化：

$$H = -\sum_{i=1}^{N} p_i \log_2(p_i)$$

| Entropy 值 | 市場狀態 | 策略 |
|-----------|---------|------|
| H < 0.3 | 趨勢明確，低雜訊 | 放大部位，順勢操作 |
| H > 0.7 | 方向不明，高雜訊 | 縮小部位或觀望 |

**跨標的 ATR 標準化**：$ATR_{norm} = \frac{ATR}{Close} \times 100$（消除價位差異，可跨商品比較波動率）

## 績效評估框架

### MAE / MFE

$$\text{MAE (long)} = \text{entry} - \min(\text{intra-trade low})$$
$$\text{MFE (long)} = \max(\text{intra-trade high}) - \text{entry}$$

$$\text{MAFE ratio} = \frac{\text{MAE}}{\text{MFE}}$$

低 MAFE ratio → 逆勢小，順勢大 → 好的風險報酬結構。

### 核心閾值

| 指標 | 目標 | 觸發重審 | 備注 |
|------|------|---------|------|
| Profit Factor | 1.5 – 2.5 | < 1.5 | PF > 3.0 通常代表樣本過小或過度擬合 |
| Expectancy | > 0 | ≤ 0 | 系統生命線：$E = (AvgWin \times Win\%) - (AvgLoss \times Loss\%)$ |
| 年度 MDD | < 30% | ≥ 30% | |
| Recovery Factor | > 1 | < 1 | 總獲利 / MaxDD：衡量「爬坑」效率 |

### 回測 vs 實盤的結構性差異

**實盤績效 = 回測績效 − 滑價 − 執行延遲**

| 因素 | 影響 | 注意點 |
|------|------|--------|
| 滑價（Slippage） | 實盤報酬 < 回測報酬 | 持倉時間越短，滑價佔比越高 |
| 執行延遲 | 高頻策略受影響最大 | 回測假設即時成交，實盤不是 |
| Overfitting 警示 | PF 過高、樣本期過短 | 用 Walk-Forward 驗證泛化 |

**Fat-tail 診斷**：策略是否依賴少數幾筆暴利生存？若是，穩定性低，隨機性高 → 需要更大樣本驗證邊緣。

## 海龜系統核心（Unit Sizing）

$$\text{1 Unit} = \frac{1\% \times \text{Account Equity}}{N \times \text{DollarPerPoint}}$$

其中 $N$ = 20日 EMA 真實波幅（ATR）。每個 Unit 的風險等於帳戶淨值的 1%。

**加碼步長**：每向有利方向移動 $0.5N$ 加 1 Unit。
**進場訊號**：突破20日高點（短期）或55日高點（長期）。

## 市場生存論

從交易哲學提煉的結構性認知，不是建議，是邊界條件：

**1. 不行動的偏好是優勢**

大部分時間市況無事可為。頂尖交易者的行動密度極低——一年幾筆大交易，其餘時間是等待和心理管理。「今天不是你找到別人忽略的機會的日子」是預設值，而不是例外。

**2. 時間的機會成本不可忽視**

```
主動交易的時間成本 = 高時薪職業的放棄收入
在資產規模不夠大之前，本業複利 >> 交易複利
```

驗算：100 萬資產 × 10% 年化 = 10 萬。同期時間在本業能賺多少？數字說話，不是信仰問題。

**3. 優勢必須在非對稱場找**

在有效市場（大型股、主要期貨）正面對抗機構=負 EV。有效市場的信息反應速度已是現代奇蹟。

可能存在優勢的地方：
- 流動性差的小市場（機構不進）
- 資訊處理速度差異（技術/速度邊緣）
- 結構性套利（非方向性）

**4. 交易的方差遠大於撲克**

撲克的方差已經讓頂尖玩家信心崩潰。交易的方差是撲克的數倍。這意味著：短期結果無法驗證你是否有邊際優勢，需要的樣本量大到一個人的職業生涯可能都不夠。

**對系統設計的影響**：
- 回測必須包含足夠長的週期和不同市場狀態
- 單一策略的樣本數不足時，不能宣稱「有邊緣」
- Profit Factor > 1.5 是門檻，不是保證

## MAEMFE 精細化分析

標準 MAEMFE 預設逆勢交易（Ricky 原始方法論，FX 為主）。應用到趨勢型商品需要調整：

**關鍵差異**：
- 逆勢交易：MFE 發生在 MAE 之前（先到達目標再承受逆境）
- 順勢交易：MFE 發生在 MAE 之後（先承受逆境再到達目標）→ 使用 Global（全域）而非 Local 定義

```
Global MAE = entry - min(intra-trade low)   # 持倉期間最大逆境
Global MFE = max(intra-trade high) - entry  # 持倉期間最大順境
```

**四象限分解**：每支策略拆成多單/空單 × 勝手/敗手，各自分析，避免混合訊號掩蓋結構問題。

**停損設置方法（從 PnL 分佈推導）**：

| 方法 | 位置 | 適用 |
|------|------|------|
| Q1（PnL count） | 避免極端損失 | 較大 SL，逆勢策略 |
| Q2（PnL count） | 篩掉大量小虧，保留少數大獲利 | 較小 SL，順勢策略 |
| Win Trade Frontier | MAE scatter 中的勝負分界線 | 最精確，但需樣本夠大 |

**體質判斷（MAE 分佈形狀）**：
- 左偏或對稱 → 健康，大部分交易逆境小
- 半峰（趨勢策略）→ 持有期間變化大，正常
- 峽峰 → 參數過度敏感，頻繁交易，需注意

**Edge-Ratio = MFE / MAE**。這個比率的分佈比單純的 NetProfit 和 MDD 更能揭示策略體質。投組多樣化時，用 Edge-Ratio 分布的相關性比用績效曲線相關性更準。

## 投組建構：貪婪演算法策略選入

單支策略績效是充分條件，互補性才是投組的核心邏輯。

**貪婪前向選入 + 後向消除**：

```
1. 找目標函數最高的單支策略 → P
2. 嘗試所有 S，找 P+S 中改善目標函數最多的 S* → 加入 P
3. 重複直到無法再改善 → P* 完成
4. 從策略池移除 P*，重複 1-3 → 產出 P1, P2...Pn
5. 等權合成 P** = P1 + P2 + ... + Pn
6. 後向消除：逐一移除策略，若能改善則移除，重複直到穩定
```

**目標函數優先序**：

| 目標函數 | 適用場景 |
|---------|---------|
| NP / MDD | 基本風報比 |
| AVG(NP) / AVG(DD) | 跨年穩定性 |
| Sharpe / Sortino | 含無風險利率 |
| 相關性（Corr） | 互補性指標 |

**選策略的邊界條件**：
- 策略體質：交易次數足夠 + 不凹單（樣本外泛化）
- 單支策略佔比不超過投組上限（策略失效隔離）
- 近3個月損益 > 0 且近6個月損益 > 0（過濾失效策略）

**核心洞察**：MDD 加碼只在策略未失效時有效 → 所以加碼邏輯必須附帶策略健康度過濾。

## 交易檢討逆向工程

每筆交易結束後的四步歸因：

1. **進場歸因** — 進場信號是否符合策略定義？（執行問題 vs 策略問題）
2. **出場歸因** — 是否按計劃執行停損/停利？（紀律問題 vs 系統問題）
3. **對手識別** — 誰賺走了這筆錢？他們的邏輯是什麼？
4. **策略同化** — 能否將對手的獲利模式納入自身策略？

**部位中性原則**：對市場方向的判斷必須獨立於當前持倉方向。
持有多單時問「如果我空著我會怎麼看？」答案不同 → 倉位中有情緒，不是信號。
