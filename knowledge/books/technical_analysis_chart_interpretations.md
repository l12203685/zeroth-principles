### TL;DR (≤120字)
一本標準的技術分析 comprehensive guide——覆蓋從 Dow Theory 到 Elliott Wave 到 Fibonacci 的全部主流技術分析工具與 chart pattern。價值在於提供完整 vocabulary 與 pattern 目錄，讓交易者能讀懂研究報告與交易員術語；但對這些工具的實際 edge（vs random chance）討論薄弱。

### 核心本質 (3-5 條)
1. **技術分析的三大核心假設**（本質） — (1) 價格 discounts all news（市場有效加強版）；(2) 價格 move in trends（非 random walk）；(3) 歷史會重複（人類行為模式穩定）。這三假設是所有 TA 工具的哲學基礎；若否認任一條，整個框架瓦解。Aronson《Evidence-Based TA》嚴格檢驗後發現大多數 pattern 的 edge 是 marginal 或 zero。
2. **Chart Pattern 的 Self-Fulfilling Prophecy**（本質） — Head & Shoulders、Double Top 等 pattern 之所以 "work"，部分原因是 enough 交易者相信並在 break neckline 時進場，使 pattern 自我實現。這是 Keynes "beauty contest" 的 microcosm——重點不是 pattern 真有預測力，而是 others 相信它有。
3. **Support & Resistance 是 Order Flow 的幾何表達**（本質） — Support = 先前大量 buy orders 的聚集點，market 接近時買方出現；Resistance 反之。突破（breakout）的意義是 order 被吃光，新 order 在新 level 聚集。這把抽象 TA 接回可驗證的 microstructure。
4. **Multi-Timeframe Confluence**（本質） — 單 timeframe 的 signal 勝率 50-55%；多 timeframe 同向 confluence（日線趨勢 + 小時 pullback + 分鐘進場）可達 65-70%。這是 Elder Triple Screen 的理論基礎。
5. **Volume Confirmation Rule**（本質） — 價格 breakout 伴隨 volume 上升才是 valid；volume 萎縮的 breakout 多為 false signal。此原則貫穿所有 TA 框架。

### 可用戰術/策略
- **Head & Shoulders Pattern Trade**：neckline 突破 + volume 擴大 → short entry；profit target = 頭部到 neckline 距離；stop = right shoulder 上方。勝率歷史約 55-65%，profit factor 1.5-2.0。
- **Bollinger Bands Mean Reversion**：price 觸 upper band + RSI > 70 → short；反之 long。只在 ranging market 有效（ADX < 25）；在 trending market (ADX > 30) 會反向傷人。
- **Fibonacci Retracement Entry**：trend confirm 後等 pullback 到 38.2%, 50%, 61.8% 任一水位 + candlestick 反轉 pattern；進場低風險、高 R:R。
- **MA Crossover Trend Filter**：EMA(50) 在 EMA(200) 之上 → bull regime，只做多；反之 bear。不是 entry signal 是 regime filter。
- **Volume Profile Trade**：識別 value area high/low (VAH/VAL)；price 接近 VAH 且 volume 下降 → fade 上方；接近 VAL 且 volume 上升 → bounce 進場。

### 盲點 / 反例 / 適用邊界
- **Publication Bias 與缺乏 backtest 嚴謹性** — 多數 TA 書用 hindsight 挑 textbook-perfect example；真實 live trading 的 pattern recognition 充滿 noise；Aronson 用數千次 test 後發現大多數 pattern 的 edge < 0.1% per trade。
- **Elliott Wave 的 degrees of freedom 問題** — Elliott Wave 有 9 個 degrees 與無數 labeling variation；事後看似精確，事前預測一致性差。多數 Elliott practitioners 跑 live trading 報酬並不出色。
- **Fibonacci 的心理學 vs 數學** — 0.618、0.382 等比例的「自然」性質是 overstated；市場多數 "Fibonacci retracement" 在 ±5% 區間都算 hit，本質是 confirmation bias。
- **未涵蓋 algorithmic trading 影響** — 當 HFT 占成交 50%+ 時，傳統技術指標的信號被 machine 提前套利；純 TA 策略的 edge 在 2010 年後顯著衰減。
- **Cognitive bias 風險** — Pattern recognition 是人類最強也最易誤用的能力；TA 容易產出 "看到想看的"，特別是無明確規則時。
