# 阿瓦隆推理方法論

> 阿瓦隆是不完整資訊下推理的沙箱。
> 所有從這裡發展的工具，都可以直接映射到交易、投資、人際判斷。

---

## 六公理

| # | 公理 | 含義 |
|---|------|------|
| 1 | **一致性** | 所有推論必須和已知事實相符，不能自相矛盾 |
| 2 | **廣泛性** | 考慮所有可能的假設，不過早排除低概率情境 |
| 3 | **精簡性** | 解釋力相同時，假設最少的模型優先（奧卡姆剃刀）|
| 4 | **優勢性** | 優先考慮對特定陣營有利的解釋 |
| 5 | **可證偽性** | 推論必須能被未來觀測推翻或支持 |
| 6 | **自我修正** | 新資訊進來後主動更新信念，不捍衛舊推論 |

**第六條是核心**：能自我修正的推理者，最終會收斂到正確答案。

## 三層邏輯架構

```
鐵邏輯（必然成立）
  → 如果 A 是壞人，這個結論無論如何都成立
  → 最高權重，直接縮小搜索空間

正邏輯（概率加分）
  → 這個行為符合壞人的期望值最大化
  → 中等權重，累積後影響信念

反邏輯（反向排除）
  → 這個行為如果是壞人做的，反而不符合 EV
  → 用來排除假說，不用來確認
```

### 遊戲場景範例

```
場景：5人局，第二次任務（2人出征），任務失敗（1 fail）
出征者：玩家 A 和 玩家 B

鐵邏輯：A 和 B 中至少有一個壞人（任務失敗 = 必然有人出 fail）
  → 直接縮小搜索空間，不需要其他資訊

正邏輯：玩家 C 在討論時強烈指控 A，但從不提 B
  → 如果 C 是壞人且 B 也是壞人，保護隊友是 EV 最大行為
  → C 是壞人的信念 +15%

反邏輯：玩家 D 主動提議「把我放進下次任務來驗證」
  → 如果 D 是壞人，主動暴露自己不符合 EV（任務失敗會直接暴露）
  → D 是壞人的信念 -20%（排除用，不等於 D 一定是好人）
```

## 區間派票法（Bayesian 信念更新）

每位玩家維護一個信念區間：

$$[P(\text{好人}), P(\text{壞人})], \quad P(\text{好人}) + P(\text{壞人}) = 1$$

每次觀測事件觸發更新：

$$P(\text{壞人} | \text{行為}) = \frac{P(\text{行為} | \text{壞人}) \cdot P(\text{壞人})}{P(\text{行為})}$$

**可觀測的觸發點**：
- 投票 for / against 隊伍
- 任務 pass / fail
- 發言內容是否和已知資訊矛盾
- 沉默（不發言也是資訊）

### 數值範例

```
初始狀態：5 人局有 2 壞人，玩家 X 的 prior P(壞人) = 2/5 = 0.40

觀測事件：玩家 X 投票反對一個「包含已知好人」的隊伍

計算：
  P(投反對 | 壞人) = 0.7  （壞人傾向阻擋好人隊伍）
  P(投反對 | 好人) = 0.3  （好人偶爾也會基於其他理由反對）

  P(投反對) = 0.7 × 0.40 + 0.3 × 0.60 = 0.28 + 0.18 = 0.46

  P(壞人 | 投反對) = (0.7 × 0.40) / 0.46 = 0.28 / 0.46 ≈ 0.61

結果：玩家 X 的壞人信念從 0.40 → 0.61
含義：一次投票行為把不確定性顯著壓縮了
```

## 關鍵洞察

**不需要「確定」，只需要「夠高的概率」**

- 壞人的最優策略：維持全局信念熵高位（製造混亂）
- 好人的最優策略：每次觀測後降低熵（製造清晰）
- 推理能力的差異，最終體現在「誰能更快壓縮不確定性」

## 公理衝突時的優先序

公理之間偶爾會矛盾。處理原則：

```
自我修正（#6）> 一致性（#1）> 可證偽性（#5）> 精簡性（#3）> 廣泛性（#2）> 優勢性（#4）
```

| 衝突情境 | 選擇 | 理由 |
|---------|------|------|
| 精簡性 vs 廣泛性 | 先廣泛，再精簡 | 過早排除假說的代價 > 多維護一個假說的成本 |
| 一致性 vs 優勢性 | 一致性優先 | 和事實矛盾的推論，再有利也是錯的 |
| 精簡性 vs 可證偽性 | 可證偽性優先 | 簡單但無法驗證的理論沒有實戰價值 |

**核心邏輯**：第六條（自我修正）永遠最高，因為它是讓其他公理不斷改進的引擎。

### 量化交易

| Avalon 概念 | 交易對應 | 說明 |
|------------|---------|------|
| 一局遊戲 | 一個交易週期 | 從建倉到平倉的完整觀測循環 |
| 每次投票/任務 | 每根 K 棒 | 一次觀測，觸發信念更新 |
| 壞人信念 | 趨勢方向信念 | P(上漲) 隨每根 K 棒更新 |
| 任務失敗 | 停損觸發 | 假說被推翻，退出並重置信念 |
| 壞人偽裝好人 | 假突破 | 價格行為「表演」趨勢，實際是反轉 |
| 鐵邏輯 | 結構性突破 | 月線趨勢改變 = 無法反駁的事實 |
