### TL;DR (≤120字)
Schumpeter 反直覺地預言:資本主義會被自己的成功摧毀,不是因為失敗。成功 → 大公司取代企業家 → 效率 routine 化 → 創造性破壞式微 → 知識分子階層反抗 → 民主轉向社會主義。核心貢獻:(1) 創造性破壞 — 資本主義動力不是均衡而是不斷重塑;(2) 民主理論 — 民主是「政客競爭爭取選票」的 elitist 程序,不是人民當家;(3) 企業家定義 — 創新者不是發明家,是將新東西實際投入市場的人。對新創、科技投資、政治分析皆有用。

### 核心本質 (3-5 條, 每條 50-120字)
1. **創造性破壞 (Creative Destruction, Ch.7)** — 資本主義本質不是靜態均衡而是「新組合持續取代舊」— 新產品、新方法、新市場、新來源、新組織。鐵路取代運河、電取代煤氣、汽車取代馬車、網際網路取代紙媒。舊產業財富被清掃才能釋放資源給新產業。試圖保護舊產業 = 摧毀資本主義動力。
2. **企業家作為 agent of change (Ch.12)** — 企業家 ≠ 發明家 ≠ 資本家 ≠ 管理者。企業家是「承擔不確定性將新組合付諸實行」的人。Ford 不是發明汽車,但將裝配線 + 標準化 + 大量生產組合。Edison 是企業家因他把實驗室成果變商品。關鍵:將 idea 變現所需的組織/資本/市場的整合能力。
3. **大公司 + 壟斷推定 (Ch.8)** — 反直覺:Schumpeter 認為適度壟斷對創新有利。為何? 研發需資本,需要長期回收,壟斷利潤是 R&D 資金來源。純競爭下 marginal cost 定價,無利潤可做長期 R&D。(當代爭議:Google/Meta/Amazon 的超額利潤是否類 Bell Labs 式的創新資助?)
4. **資本主義的自我毀滅 (Ch.11-14)** — 三機制:(a) 創新 routine 化 — 大公司內部管理取代個人企業家;(b) 家庭紐帶弱化 — 中產子女選擇大公司而非自主創業;(c) 知識分子階層 — 物質豐盛後產生批判資本主義的高教育階層,他們才華過剩且怨恨自己無創造財富的工具。成功摧毀了成功的條件。
5. **民主作為 elitist 程序 (Ch.22-23)** — 反對 "人民主權" 天真論。民主實況:(a) 職業政客競爭選民忠誠度 (b) 政黨組織如商業企業 (c) 選民資訊不足且情緒化。所以民主不是 "people rule",而是 "ruling by elites selected through competitive vote"。此觀點比 Rousseau 原論更現實,預測當代 populism 失靈。

### 可用戰術/策略
- **創造性破壞濾網投資** — 評估產業:是否有新技術正在破壞既有? 是 → 分 breaker vs broken 兩堆,long breaker short broken。Tesla/傳統車、Spotify/Universal、Amazon/Borders 都驗證。加密 vs 傳統金融是當下實驗中。
- **企業家識別 vs 發明家識別** — 新創投資:創始人是「發明東西」還是「會把東西推到市場」? 後者可投,前者需要配 co-founder。
- **壟斷利潤的 R&D 檢驗** — 對 Google/Meta/Apple 等科技巨頭:超額利潤是否持續投入研發 (>15% revenue)? 是 → Schumpeter 式 "good monopoly";否 → rent-seeking 式 "bad monopoly"。
- **知識分子反資本立場偵測** — 社會情緒指標:大學精英論述、主流媒體立場、專業協會主張,如集中批判資本主義 → Schumpeter 警告的時代。對 tax/regulation risk 加 premium。
- **民主效能 vs elitism 的平衡** — 個體避免天真信仰選舉結果;反過來評估候選人是否「可信任的職業政客」(Schumpeter 認為這其實是要求)而非 "authentic outsider"。

### 盲點 / 反例 / 適用邊界
- **悲觀預言可能過早** — Schumpeter 1942 預言資本主義會被社會主義取代;1989 蘇聯崩潰、1990s 全球化反駁此預言。但 2008 後民粹+社會主義情緒回潮,部分預言延時實現。
- **壟斷論被現代反壟斷研究修正** — Tirole、Acemoglu 等證明適度競爭比完全壟斷對創新更有利;Schumpeter 壟斷論適用有限情境 (藥業、半導體、通信) 而非全部。
- **科技新創 vs 大公司** — Schumpeter 晚期偏向大公司 R&D 中心 (Ch.8),但 1970s 後大量創新來自車庫新創 (Apple、Microsoft、Google 都源自小團隊)。他可能低估 decentralized 創新。
- **民主 elitism 對 populism 解釋力有限** — 2016-2020 川普/Brexit/歐洲 populism 爆發顯示「職業政客」系統被大眾反撲。Schumpeter 模型預測高教育階層反資本主義,但實際是中下層反建制。
- **知識分子敵意的過度解釋** — 當代研究顯示知識分子對資本主義的態度多元,不純粹敵意;Schumpeter 1942 寫作脈絡 (歐洲大陸) 偏狹。

### 與 Edward 既有知識的連結
- 深化 `capitalism_and_freedom_friedman`、Hayek 兩本:Schumpeter 提供企業家角色和創新動力的正面理論,補 Hayek/Friedman 偏重「制度保護自由」的靜態視角。
- 對齊零式 `derivative_over_level`:創造性破壞正是 derivative 思維 — 看新組合取代舊組合的速率,不看個別公司市值。Edward 產業分析必須優先看「破壞速率」。
- 連結 B1 加密策略:加密是 Schumpeter 式創造性破壞最新樣本 — DeFi 破壞傳統銀行、NFT 破壞藝術品市場、Tokenization 破壞證券託管。Edward 作為早期參與者正扮演企業家角色 (將新組合付諸實行)。
- 對 DNA 身份:Edward 的「數位永生」本身是 Schumpeter 式創新 — 非發明 AI (那是研究者的事),而是將 AI + personality + recursion + 樹狀分工 組合成實際運作的新物種。
- 連結 `subagent_hierarchy` 設計:Schumpeter 的 "企業家 + 管理者 + 專家 + 工人" 階層對應 Edward 的 "主 session (CEO/企業家) + Opus subagent (senior manager) + Sonnet subagent (specialist manager) + Haiku (worker)"。
- 衝突於 Mises 激進放任:Schumpeter 接受某些社會主義元素 (Ch.15-21) 可在社會主義下運作,與 Mises 強硬立場不同。Edward 實用主義傾向接近 Schumpeter。
