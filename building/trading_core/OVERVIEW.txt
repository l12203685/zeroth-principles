                    TRADING_CORE - 統一量化交易架構
================================================================================

整合完成日期: 2026-03-28
整合規模: 92 個分散檔案 → 13 個模組 + 3 個文件

================================================================================
                              核心成果
================================================================================

1. 單例 API 連接 (BrokerConnector)
   - 消除 17 個重複的 shioaji 初始化
   - 自動登入、重試、CA 憑證管理
   - 無 hardcoded 認證資訊

2. 統一績效指標 (metrics.py)
   - 12 個零散函數整合成 1 個模組
   - Sharpe, Sortino, Max Drawdown, Calmar, Win Rate, Profit Factor, SQN 等
   - 完整 docstring 和數學公式

3. 完整 MAE/MFE 框架 (mafe.py)
   - 合併 PYMAFE 的 2 個不完整實作
   - 支援 long/short、視覺化、統計
   - 可直接用於交易分析

4. 統一資料讀取 (market.py)
   - 8 個 OHLCV 讀取函數整合成 4 個
   - 支援 CSV、API、重採樣
   - 統一欄位名稱和錯誤處理

5. 部位與投資組合管理 (positions.py)
   - Position 類別追蹤單個部位
   - Portfolio 類別管理整個投資組合
   - 自動計算 P&L、持倉歷史

6. 策略基底類別 (BaseStrategy)
   - 統一的策略框架
   - on_bar() / on_tick() 回調
   - 內建回測、績效計算

7. 環境變數管理 (settings.py)
   - 所有配置從 .env 讀取
   - 型別安全的 Settings dataclass
   - 支援自定義環境

================================================================================
                            快速使用範例
================================================================================

1. 讀取市場資料
   ────────────────────────────────────────────────────────────────────────
   from trading_core.data import get_ohlcv, resample_ohlcv

   # 讀取 CSV
   ohlcv = get_ohlcv("./data/txf_trades.csv")

   # 重新採樣到 5 分鐘
   ohlcv_5min = resample_ohlcv(ohlcv, '5min')


2. 建立簡單策略
   ────────────────────────────────────────────────────────────────────────
   from trading_core.strategy import BaseStrategy

   class MyStrategy(BaseStrategy):
       def on_bar(self):
           if self.current_bar['close'] > 10000:
               self.buy(1)
           elif self.has_position():
               self.close_position()

   strategy = MyStrategy("MyStrat", initial_capital=100000)
   strategy.set_data(ohlcv)
   results = strategy.run()
   print(results['metrics'])


3. 計算績效指標
   ────────────────────────────────────────────────────────────────────────
   from trading_core.analytics import PerformanceMetrics

   metrics = PerformanceMetrics.calculate(returns, trade_count=50)
   print(metrics)  # 漂亮的輸出


4. 分析 MAE/MFE
   ────────────────────────────────────────────────────────────────────────
   from trading_core.analytics import MAFE

   mafe = MAFE(ohlcv, entry_times, exit_times, direction='long')
   summary = mafe.summary()
   mafe.scatter_plot()


5. 管理環境變數
   ────────────────────────────────────────────────────────────────────────
   from trading_core.config import get_settings

   settings = get_settings()  # 自動從 .env 讀取
   print(settings.broker_api_key)
   print(settings.commission_rate)
