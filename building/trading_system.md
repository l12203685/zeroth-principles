
## 市場狀態機

市場有三種狀態，對應不同操作模式：

| 狀態 | 觸發條件 | 操作 |
|------|---------|------|
| 攻擊 | 月線+季線多頭 + ATR 低 | 高 Beta 加碼 |
| 防禦 | 月線空頭 + ATR 高 | 避險 / CTA |
| 中性 | 訊號模糊 | 等權持倉，等待清晰 |

**「不確定的時候，站回中性。」**

### 狀態轉換邏輯

```
攻擊 → 中性：月線或季線任一轉空，或 ATR 突破 20日均值 1.5 倍
中性 → 攻擊：月線+季線同時確認多頭，且 ATR 回落至均值以下
中性 → 防禦：月線確認空頭 + ATR 持續擴大超過 2 週
防禦 → 中性：ATR 開始收斂 + 價格不再創新低
防禦 → 攻擊：不直接跳轉，必須經過中性（防止反彈陷阱）
```

**設計原則**：防禦 → 攻擊不允許直接跳轉。急跌後的反彈最容易讓人誤判，強制經過中性是一個結構性保護。

### 策略 × 市場狀態對應

| 策略 | 攻擊 | 中性 | 防禦 |
|------|------|------|------|
| ORB | ✅ 主力策略 | ❌ 停用 | ❌ 停用 |
| Resonance | ✅ 適用 | ⚠️ 減倉使用 | ❌ 停用 |
| Prime K | ⚠️ 輔助確認 | ✅ 主力策略 | ⚠️ 反向使用 |
| Tofu | ❌ 機會成本高 | ✅ 適用 | ✅ 適用 |

## 績效評估框架

### MAE / MFE

$$\text{MAE (long)} = \text{entry} - \min(\text{intra-trade low})$$
$$\text{MFE (long)} = \max(\text{intra-trade high}) - \text{entry}$$

$$\text{MAFE ratio} = \frac{\text{MAE}}{\text{MFE}}$$

低 MAFE ratio → 逆勢小，順勢大 → 好的風險報酬結構。

### 數值範例

```
一筆多單交易：
  進場價：100
  交易期間最低價：97（最大逆行）
  交易期間最高價：112（最大順行）
  出場價：108

  MAE = 100 - 97 = 3（你承受的最大浮虧）
  MFE = 112 - 100 = 12（你曾經擁有的最大浮盈）
  MAFE ratio = 3 / 12 = 0.25（優秀，逆勢遠小於順勢）

  對比一筆差的交易：
  進場價：100，最低：93，最高：103，出場：99
  MAE = 7, MFE = 3, MAFE = 2.33（糟糕，逆勢遠大於順勢）

解讀：
  MAFE < 0.5 → 進場位置好，風險報酬結構健康
  MAFE > 1.0 → 進場位置差，在逆勢中掙扎
  系統性 MAFE > 1.0 → 策略本身有問題，不是運氣
```

### 核心閾值

| 指標 | 目標 | 觸發重審 |
|------|------|---------|
| Profit Factor | 1.5 – 2.5 | < 1.5 |
| Expectancy | > 0 | ≤ 0 |
| 年度 MDD | < 30% | ≥ 30% |

## 策略矩陣

| 策略 | 核心邏輯 | 適用市況 |
|------|---------|---------|
| ORB | 開盤區間突破，捕捉早盤動能 | 趨勢日 |
| Resonance | 多週期趨勢共振入場 | 中期趨勢明確 |
| Prime K | 關鍵K棒結構判斷主力方向 | 盤整後突破 |
| Tofu | 窄幅區間來回收割 | 低波動橫盤 |

### 策略細節

**ORB（Opening Range Breakout）**
- 進場：開盤 15-30 分鐘高低點確立後，突破方向進場
- 出場：收盤前平倉（日內策略），或觸及 2R 目標
- 風控：停損設在開盤區間的另一端，ATR < 20日均值時不進場
- 關鍵：趨勢日辨識率決定這個策略的 PnL

**Resonance（多週期共振）**
- 進場：日線、週線、月線趨勢方向一致時，在日線回撤支撐進場
- 出場：週線趨勢翻轉，或達到月線級別的阻力/支撐
- 風控：停損在日線結構破壞位，部位不超過總資金 5%
- 關鍵：耐心等共振，不在單一週期信號就進場

**Prime K（關鍵K棒）**
- 進場：識別帶量長實體 K 棒（主力表態），在其收盤價或 50% 回撤進場
- 出場：後續 K 棒未能延續方向則出場，或達到 Prime K 等幅目標
- 風控：停損在 Prime K 的低點（多單）或高點（空單）
- 關鍵：辨別真正的主力 K 棒 vs 假突破 K 棒，量是核心過濾器

**Tofu（豆腐策略）**
- 進場：價格在窄幅區間（ATR < 20日均值 × 0.6）內，碰觸區間邊緣反向進場
- 出場：到達區間另一端，或區間被突破則停損
- 風控：嚴格限制在低波動環境，區間突破立刻認錯
- 關鍵：紀律 > 判斷，這是機械化程度最高的策略
